# Compare NLP models using [lime](https://github.com/marcotcr/lime)

Starlette web application for dynamically trying lime on 4 chosen models:
* [tfidf](https://github.com/yonigottesman/nlp_lime/blob/master/notebooks/tfidf_linear.ipynb)
* [fasttext](https://github.com/yonigottesman/nlp_lime/blob/master/notebooks/fasttext.ipynb)
* [rnn](https://github.com/yonigottesman/nlp_lime/blob/master/notebooks/rnn_pytorch.ipynb)
* [transformer](https://github.com/yonigottesman/nlp_lime/blob/master/notebooks/bert.ipynb)
